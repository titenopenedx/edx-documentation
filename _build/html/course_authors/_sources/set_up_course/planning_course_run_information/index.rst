################################################################
Planning Course Run Information (edx.org and Edge Courses)
################################################################

Course run information, such as start and end dates, is specific to a course
run. This section lists the optional and required information for course runs,
as well as guidelines to help you create a successful course run.

After you have determined the information for your course run, you enter this
information in Publisher, Studio, or both. For more information, see
:ref:`Introduction to Publisher` or :ref:`Partner Add Studio Information`.
