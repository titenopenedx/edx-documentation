######################################################
Planning Course Information (edx.org and Edge Courses)
######################################################

Course information remains true whether the course runs one time or multiple
times. This section lists the optional and required information for courses, as
well as guidelines to help you create an effective course and About page.

After you have determined the information for your course, you enter this
information in Publisher, Studio, or both. For more information, see
:ref:`Introduction to Publisher` or :ref:`Partner Add Studio Information`.
