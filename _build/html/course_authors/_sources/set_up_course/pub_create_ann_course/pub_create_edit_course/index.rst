##############################
Creating a Course in Publisher
##############################

.. note::
  This process applies to courses on the edx.org site. If your course will run
  on Edge, you create the course and the About page in Studio. For more
  information, see :ref:`Creating a New Course` or :ref:`Creating a Course
  About Page`.

This section describes the process of creating a course in Publisher. This
section also includes information about finding a course that has been created,
adding a comment to a course, and other actions.
